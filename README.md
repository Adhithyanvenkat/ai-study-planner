# 🤖 AI Study Planner

AI Study Planner is a smart web application developed to help students organize and manage their study schedules efficiently. The system uses AI-based recommendations to create personalized study plans based on subjects, priorities, deadlines, and available study time.

---

## 📌 Features

- 📅 Personalized Study Timetable
- ⏰ Smart Time Management
- 📚 Subject & Task Prioritization
- 🔔 Study Reminders
- 📊 Progress Tracking
- 🤖 AI-Based Learning Recommendations
- 🌐 Responsive User Interface
